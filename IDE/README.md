

# Habit Tracker

**Habit Tracker** — это консольное приложение на Python для отслеживания ежедневных привычек. Помогает формировать полезные привычки и контролировать их выполнение с помощью системы непрерывных серий (streak).

## Возможности

- ✅ **Создание привычек** — добавление новых привычек с названием и описанием
- ✅ **Отслеживание выполнения** — отметка выполнения привычки за текущий день
- ✅ **Подсчёт серии (streak)** — автоматическое увеличение счётчика при ежедневном выполнении
- ✅ **Автоматический сброс серии** — если пропущен день, серия сбрасывается до нуля
- ✅ **Редактирование** — изменение названия и описания привычки
- ✅ **Сохранение данных** — все данные автоматически сохраняются в JSON-файл
- ✅ **Кроссплатформенность** — работает на Windows, macOS и Linux

## Технологии

- **Python 3.7+** — язык программирования
- **JSON** — формат хранения данных
- **Git** — система контроля версий

## Установка и запуск

### Требования

- Установленный Python версии 3.7 или выше
- Git (опционально, для клонирования репозитория)

### Инструкция по установке

1. **Клонируйте репозиторий**
   ```bash
   git clone https://github.com/your-username/habit-tracker.git
   cd habit-tracker
   ```

2. **Запустите приложение**
   ```bash
   python main.py
   ```

   > **Примечание:** На некоторых системах может потребоваться использовать `python3` вместо `python`

### Альтернативный способ (без Git)

1. Скачайте все файлы проекта в одну папку
2. Откройте терминал в этой папке
3. Выполните команду:
   ```bash
   python main.py
   ```

## Использование

### Главное меню

При запуске приложения отображается главное меню:

```
=== Habit Tracker ===
1. Показать привычки
2. Добавить привычку
3. Отметить выполнение
4. Выйти
5. Редактировать привычку
```

### Основные действия

#### 1️⃣ Показать привычки
Отображает список всех привычек с их текущей серией (streak) и описанием.

#### 2️⃣ Добавить привычку
Запрашивает название и описание новой привычки, затем добавляет её в список.

#### 3️⃣ Отметить выполнение
Позволяет отметить привычку как выполненную сегодня. Серия (streak) автоматически увеличивается на 1.

#### 4️⃣ Выйти
Завершает работу приложения. Все данные автоматически сохраняются.

#### 5️⃣ Редактировать привычку
Позволяет изменить название и/или описание существующей привычки.

### Особенности работы

- **Пропуск дня:** Если вы не отметили выполнение привычки в течение дня, при следующем запуске серия автоматически сбросится до нуля.
- **Сохранение данных:** Все привычки сохраняются в файл `habits.json` в той же папке, где находится программа.

## Структура проекта

```
habit-tracker/
│
├── main.py                 # Главный модуль, точка входа в приложение
├── habit.py               # Класс Habit (привычка)
├── storage.py             # Функции для сохранения/загрузки данных
├── utils.py               # Вспомогательные функции
├── streak_checker.py      # Проверка и сброс серий
├── habits.json            # Файл с данными (создаётся автоматически)
├── .gitignore             # Исключаемые файлы для Git
└── README.md              # Документация проекта
```

### Описание модулей

| Файл | Назначение |
|------|------------|
| `main.py` | Основной цикл программы, обработка пользовательского ввода |
| `habit.py` | Определение класса Habit (название, описание, streak, даты выполнения) |
| `storage.py` | Чтение и запись данных в JSON-файл |
| `utils.py` | Утилиты: получение текущей даты, отображение списка привычек |
| `streak_checker.py` | Логика проверки пропущенных дней и сброса streak |

```

Номер привычки: 1
Отмечено!
```
